# Web Application SATECHO

Frontend application developed with Vue 3 + Vite following DDD architecture and GitFlow workflow.

## Technologies

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios
- Sass

## Architecture

The project follows:

- Domain-Driven Design (DDD)
- Bounded Contexts
- GitFlow
